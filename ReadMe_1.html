TinyXML

Introdução

O TinyXML é uma pequena biblioteca C++ multi-plataforma que implementa um parser simples de XML, que permite, entre outras funcionalidades:

  • a leitura de um documento XML para uma estrutura interna de dados constituída por nós e elementos (Nodes e Elements) XML,
  • o acesso aos dados de diversas formas: iterativamente ou através de acesso directo a determinados elementos
  • a alteração da estrutura, inserindo, modificando ou removendo elementos
  • a gravação para ficheiro em formato XML de uma estrutura em memória

Instalação

Para utilizar o TinyXML em projectos simples em C++/win32, apenas são necessários os seguintes ficheiros: os binários da biblioteca tinyxml.lib (versão release) ou tinyxmld.lib (versão debug) e dois header files, tinyxml.h e tinystr.h.

Estes ficheiros podem ser colocados nas respectivas directorias "lib" e "include" do sistema, tal como indicado para as bibliotecas de GLUT (ver documentação de instalação de GLUT) ou, alternativamente, podem ser colocados na própria pasta do projecto, sendo necessário incluir a lib no próprio projecto.

Os ficheiros referidos podem ser obtidos de duas formas:

  • Utilizando a versão disponibilizada no FTP, baseada nas sources com a versão 2.5.3 e compiladas com VS 2008
  • Compilando as próprias versões, utilizando as sources disponíveis no endereço http://sourceforge.net/projects/tinyxml/. O pacote inclui os headers referidos, e um projecto de Visual Studio que ao ser compilado gera os ficheiros ".lib" referidos (tinyxml.lib ou tinyxmld.lib, conforme o perfil de compilação escolhido, release ou debug).

Utilização do exemplo junto

  • A bilbioteca (TinyXML.lib) encontra-se incluída no projecto e é referida no código,
  • Inclui-se o header tinyxml.h (#include "tinyxml.h")
  • No main, invoca-se LoadScene()
  • Em SceneLoader, lê-se o XML e imprime-se o resultdo na consola.

O XML encontra-se no directório do projecto.
